# NotebookLM Video Package: Start With AI Today

## Files

| File | Description |
|------|-------------|
| `01-essay.md` | Source essay: "Stop Overthinking It: A Working Professional's No-Nonsense Guide to Starting With AI Today" |
| `02-prompt-engineering.md` | Knowledge base: Prompt Engineering -- techniques for effective prompts |
| `03-the-observer-mindset.md` | Knowledge base: The Observer Mindset -- the pattern of watching from the sidelines |
| `04-three-week-cliff.md` | Knowledge base: The Three-Week Cliff -- why 80% of users abandon AI tools |
| `notebooklm-prompt.md` | Custom prompt for NotebookLM video generation |
| `youtube-description.md` | Ready-to-paste YouTube video description |

## Workflow

1. Go to [NotebookLM](https://notebooklm.google.com/)
2. Create a new notebook
3. Upload `01-essay.md`, `02-prompt-engineering.md`, `03-the-observer-mindset.md`, `04-three-week-cliff.md` as sources
4. Generate a Video Overview using the prompt in `notebooklm-prompt.md`
5. Review and download the generated video
6. Upload to YouTube using `youtube-description.md` for the description
7. Fill in the timestamp placeholders (0:00) with actual timestamps

## Video Angle

Approachable, encouraging tone. "Your colleague who already figured this out explaining it over coffee." Target: 6-8 minutes.
