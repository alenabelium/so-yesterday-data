# NotebookLM Video Package: The Last Hands on the Keyboard

## Files

| File | Description |
|------|-------------|
| `01-essay.md` | Source essay: "The Last Hands on the Keyboard: How Software Development Became Agent Orchestration" |
| `02-agent-orchestration.md` | Knowledge base: Agent Orchestration -- managing multiple AI agents for complex tasks |

| `youtube-description.md` | Ready-to-paste YouTube video description |

## Workflow

1. Go to [NotebookLM](https://notebooklm.google.com/)
2. Create a new notebook
3. Upload `01-essay.md`, `02-agent-orchestration.md`, `03-specification-quality.md`, `04-ai-coding-levels.md`, `05-blast-radius.md` as sources
4. Generate a Video Overview using the prompt in `notebooklm-prompt.md`
5. Review and download the generated video
6. Upload to YouTube using `youtube-description.md` for the description
7. Fill in the timestamp placeholders (0:00) with actual timestamps

## Video Angle

Technical but accessible. The orchestra conductor analogy. Three layers of value. Five core orchestration skills. Target: 8-10 minutes.
