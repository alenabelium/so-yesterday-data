## Workflow

1. Open [NotebookLM](https://notebooklm.google.com)
2. Create new notebook
3. Upload files 01-06 as sources
4. Open Audio Overview or Video Overview
5. Paste the prompt from `notebooklm-prompt.md` into the customization field
6. Generate → review → iterate if needed
7. Download the output
8. Upload to YouTube
9. Update the essay to include the YouTube link

## After YouTube upload

Add this line after the essay's opening blockquote:

```markdown
> *Watch the [video deep dive](https://youtube.com/watch?v=VIDEO_ID) exploring these ideas.*
```
