# NotebookLM Video Package: The Intelligence Economy

## Files

| File | Description |
|------|-------------|
| `01-essay.md` | The full essay — primary source for the video |
| `02-ai-native-economics.md` | Knowledge page: AI-native company economics (5-7x revenue/employee) |
| `03-solo-operator.md` | Knowledge page: Solo operators building at company scale |
| `04-token-economics.md` | Knowledge page: Token pricing and cost structures |
| `05-ai-infrastructure.md` | Knowledge page: Physical and cloud infrastructure powering AI |
| `notebooklm-prompt.md` | Custom prompt for NotebookLM Video Overview generation |
| `youtube-description.md` | Ready-to-paste YouTube video description |

## Workflow

1. Open [NotebookLM](https://notebooklm.google.com/) and create a new notebook
2. Upload `01-essay.md` through `05-ai-infrastructure.md` as sources
3. Go to the Audio/Video Overview section
4. Paste the contents of `notebooklm-prompt.md` as the custom prompt
5. Generate the Video Overview
6. Download the generated video
7. Upload to YouTube using `youtube-description.md` for the description
8. Fill in the `XX:XX` timestamp placeholders after reviewing the video

## Video Details

- **Target length:** 8-10 minutes
- **Tone:** Business strategy — authoritative but accessible
- **Angle:** The shift from platform economics to intelligence economics, with case studies
- **Hook:** "Why the biggest business model shift since Uber is already here."
