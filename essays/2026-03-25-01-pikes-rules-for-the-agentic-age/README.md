# NotebookLM Video Package: Pike's Rules for the Agentic Age

## Files

| File | Description |
|------|-------------|
| `01-essay.md` | The full essay — primary source for the video |
| `02-agent-orchestration.md` | Knowledge page: Managing multiple AI agents in parallel |
| `03-specification-quality.md` | Knowledge page: Writing precise specs for AI systems |
| `04-blast-radius.md` | Knowledge page: Limiting damage from agent errors |
| `05-evals.md` | Knowledge page: Domain-specific evaluation frameworks for agents |
| `notebooklm-prompt.md` | Custom prompt for NotebookLM Video Overview generation |
| `youtube-description.md` | Ready-to-paste YouTube video description |

## Workflow

1. Open [NotebookLM](https://notebooklm.google.com/) and create a new notebook
2. Upload `01-essay.md` through `05-evals.md` as sources
3. Go to the Audio/Video Overview section
4. Paste the contents of `notebooklm-prompt.md` as the custom prompt
5. Generate the Video Overview
6. Download the generated video
7. Upload to YouTube using `youtube-description.md` for the description
8. Fill in the `XX:XX` timestamp placeholders after reviewing the video

## Video Details

- **Target length:** 10-12 minutes
- **Tone:** Intellectual but practical — "timeless wisdom meets cutting-edge tech"
- **Angle:** Show the five rules from 1989, then reveal how each maps to agentic engineering. Practical framework with 5 phases.
- **Hook:** Rules written for Bell Labs C programmers are the best guide for 2026 AI engineering.
