# NotebookLM Video Package: 72 Days That Rewrote the Rules

## Files

| File | Description |
|------|-------------|
| `01-essay.md` | Source essay: "72 Days That Rewrote the Rules: What Actually Happened in AI Since January 2026" |
| `02-job-market-bifurcation.md` | Knowledge base: Job Market Bifurcation -- the split into AI-augmented and traditional roles |
| `03-scaling-laws.md` | Knowledge base: Scaling Laws -- why AI performance improves predictably with compute |
| `04-frontier-labs.md` | Knowledge base: Frontier Labs -- the leading AI organizations driving the race |
| `notebooklm-prompt.md` | Custom prompt for NotebookLM video generation |
| `youtube-description.md` | Ready-to-paste YouTube video description |

## Workflow

1. Go to [NotebookLM](https://notebooklm.google.com/)
2. Create a new notebook
3. Upload `01-essay.md`, `02-job-market-bifurcation.md`, `03-scaling-laws.md`, `04-frontier-labs.md` as sources
4. Generate a Video Overview using the prompt in `notebooklm-prompt.md`
5. Review and download the generated video
6. Upload to YouTube using `youtube-description.md` for the description
7. Fill in the timestamp placeholders (0:00) with actual timestamps

## Video Angle

Fast-paced news recap energy. "Here's what happened and why it matters." Show the data: entry-level hiring down 67%, CS placements 89% to 19%, $285B sell-off. Target: 8-10 minutes.
