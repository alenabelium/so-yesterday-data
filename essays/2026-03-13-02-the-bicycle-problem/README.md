# NotebookLM Video Package: The Bicycle Problem

## Files

| File | Description |
|------|-------------|
| `01-essay.md` | Source essay: "The Bicycle Problem: Why Going Slower Won't Save Your Career in the Age of AI" |
| `02-identity-threat.md` | Knowledge base: Identity Threat -- when AI capabilities threaten professional identity |
| `03-permission-gap.md` | Knowledge base: The Permission Gap -- waiting for external validation before adopting AI |
| `04-compounding-gap.md` | Knowledge base: The Compounding Gap -- why the advantage widens exponentially |
| `05-three-week-cliff.md` | Knowledge base: The Three-Week Cliff -- why 80% of users abandon AI tools |
| `notebooklm-prompt.md` | Custom prompt for NotebookLM video generation |
| `youtube-description.md` | Ready-to-paste YouTube video description |

## Workflow

1. Go to [NotebookLM](https://notebooklm.google.com/)
2. Create a new notebook
3. Upload `01-essay.md`, `02-identity-threat.md`, `03-permission-gap.md`, `04-compounding-gap.md`, `05-three-week-cliff.md` as sources
4. Generate a Video Overview using the prompt in `notebooklm-prompt.md`
5. Review and download the generated video
6. Upload to YouTube using `youtube-description.md` for the description
7. Fill in the timestamp placeholders (0:00) with actual timestamps

## Video Angle

Empathetic but urgent. The three psychological walls. "The skills that actually matter." Target: 8-10 minutes.
