# Salesforce Killed The Browser. Every Agent Runs Your CRM Now.

## Executive Summary

Nate Jones introduces a five-question filter for evaluating the flood of AI agent launches: does it plug into existing tools, does it let other agents build on top, does it own relevant data, is an ecosystem forming, and can you stack agents on it? He applies this filter to five representative releases -- ChatGPT Workspace Agents, Salesforce Headless 360, Microsoft Copilot Wave 3, Kimi K 2.6, and Perplexity Personal Computer -- and argues the agent race has shifted from model quality to infrastructure. His key insight is that AI tool selection is no longer a switching question but a layering question, with Anthropic increasingly appearing as an embedded engine inside other companies' products (Salesforce, Microsoft, Perplexity) rather than only a standalone offering.

## Key Points

- Salesforce Headless 360 scores highest on the five-question filter by exposing every major platform capability as APIs, MCP tools, and CLI commands -- building 60+ MCP tools and 30+ coding skills so that any agent (Claude Code, Cursor, Codex) can operate inside CRM data without a human clicking through the browser UI. [05:36](https://www.youtube.com/watch?v=dQK_pTXrGDk&t=336)
- Anthropic's enterprise strategy is increasingly to be the agent layer inside other companies' products: Claude Sonnet 4.5 is the default model in Salesforce AgentForce 5, Claude Opus 4.7 is the default orchestrator in Perplexity Computer, and Anthropic tech powers Microsoft Copilot Co-work -- making "when should I switch to Claude" the wrong question. [17:56](https://www.youtube.com/watch?v=dQK_pTXrGDk&t=1076)
- The agent market is moving toward layering rather than switching: assign research-heavy deliverables to Perplexity, Microsoft-native work to Copilot, CRM operations to Salesforce's headless layer, coding to specialist environments, and repeatable cross-tool workflows to Workspace Agents -- the expensive thing is not having multiple tools but routing work incorrectly. [15:39](https://www.youtube.com/watch?v=dQK_pTXrGDk&t=939)
