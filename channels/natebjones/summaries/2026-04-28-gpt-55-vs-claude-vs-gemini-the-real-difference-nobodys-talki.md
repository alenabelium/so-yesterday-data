# GPT-5.5 vs Claude vs Gemini: The Real Difference Nobody's Talking About

## Executive Summary

Nate Jones argues that GPT-5.5 is the strongest model in the world based on three custom hard tests: a 23-deliverable executive launch package (Dingo), a 465-file dirty data migration (Splash Brothers), and an interactive 3D NASA visualization (Artemis 2). GPT-5.5 scored 87.3 on Dingo versus Opus 4.7's 67.0, was the first model to catch planted fake customers in the data migration, and proved strong on information density but weaker on visual taste compared to Opus. Jones concludes the future is multi-model routing: 5.5 in Codex for complex multi-step execution, Opus for blank-canvas visual taste and planning, and Images 2.0 for generating reference mockups that 5.5 then implements faithfully.

## Key Points

- GPT-5.5 produced all 23 required deliverables as real artifact types in the Dingo executive test (real PowerPoint with 17 slides, spreadsheets with formulas, working dashboard), scoring 87.3 versus Opus 4.7's 67.0, while correctly treating a legally sensitive exotic pet product launch with appropriate regulatory caution. [08:51](https://www.youtube.com/watch?v=9aIYhjeYxzM&t=531)
- In the Splash Brothers data migration, GPT-5.5 was the first frontier model to catch planted fake customers (Mickey Mouse, Test Customer, ASDF) and the fake $25,000 payment, but still missed service code conflicts, left payment statuses unnormalized, and showed some backend regression compared to 5.4. [14:02](https://www.youtube.com/watch?v=9aIYhjeYxzM&t=842)
- The practical routing framework: use 5.5 in Codex as the default for complex multi-step execution and data work with validation, Opus 4.7 for blank-canvas front-end taste and planning, and Images 2.0 to generate visual references that 5.5 implements -- because inventing taste is hard but implementing to a target is much easier. [24:47](https://www.youtube.com/watch?v=9aIYhjeYxzM&t=1487)
