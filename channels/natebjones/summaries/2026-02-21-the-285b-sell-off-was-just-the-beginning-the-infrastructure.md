# The $285B Sell-Off Was Just the Beginning — The Infrastructure Story Is Bigger.

## Executive Summary

The web is forking into two parallel layers: the human web (HTML, fonts, scroll animations) and an emerging agent web (APIs, structured data, markdown, payment protocols, execution environments). In a single week, Coinbase launched agentic wallets (50M+ machine-to-machine transactions via X402), Cloudflare shipped automatic markdown conversion for any AI agent requesting a page (covering ~20% of the web), and OpenAI published tools for agent skills, shell access, and context compaction. These primitives are snapping together to create autonomous economic actors -- agents that can search, read content, pay for services, and execute tasks without human intervention.

## Key Points

- Stripe rebuilt its fraud detection from scratch because decades of ML calibrated on human shopping behavior (mouse movement, browsing time, device fingerprinting) became useless when the buyer is software; brands including Urban Outfitters, Etsy, and Coach are already onboarding agent commerce.
- Cloudflare's llms.txt and AI index standards function as the agent equivalent of robots.txt, letting sites make content discoverable to agents while including built-in X402 monetization -- creating an economic layer where agents pay to access content, potentially bypassing Google search entirely.
- The gap between infrastructure being built (fully autonomous agents with wallets, search, and execution environments) and the trust people are willing to extend to agents is the central tension of the next few years; the security community treats agents as potential adversaries, not trusted employees, with sandbox isolation at every layer.
