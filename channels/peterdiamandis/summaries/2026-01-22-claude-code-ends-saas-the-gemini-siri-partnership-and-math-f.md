# Claude Code Ends SaaS, the Gemini + Siri Partnership, and Math Finally Solves AI 

## Executive Summary

In this WTF Just Happened in Tech episode, Diamandis and co-hosts cover three major developments: Claude Opus 4.5 emerging as the world's best coding model, Google's Gemini partnering with Apple to power Siri, and breakthroughs in mathematical reasoning by AI. The panel discusses how Claude Code and similar tools are threatening the traditional SaaS model by enabling individuals to build custom software instead of subscribing to pre-built services. The Gemini-Siri partnership is framed as transforming the search box from an information tool into an action-oriented agent, potentially making websites obsolete. The hosts also examine how AI-powered vertical integration is reshaping entire industries, with companies like Tesla bypassing traditional layered supply chains to build end-to-end from raw materials.

## Key Points

- Claude Opus 4.5 represents a paradigm shift in AI coding capability, with developers describing it as unbelievable; the implication is that SaaS companies face existential threat as individuals can now build custom tools via AI coding agents rather than paying for generic subscription software.
- Google powering Siri with Gemini changes the competitive landscape fundamentally, moving from a search-box paradigm to an action-oriented agent paradigm where AI executes tasks directly rather than returning links, potentially rendering traditional websites irrelevant.
- AI is enabling radical vertical integration across industries, collapsing traditional multi-layer supply chains; exponential organizations that can pivot constantly will thrive, while rigid companies with siloed structures will struggle to compete in the new landscape.
