# Z-Image Base is out! Best local AI image model

## Executive Summary

The video reviews the newly released Z-Image Base model, highlighting its superior diversity and support for negative prompts compared to the faster but less varied Z-Image Turbo. It provides a comprehensive comparison against Flux 2, demonstrating Z-Image's strength in character recognition and artistic style adherence, while noting its slower generation speed and occasional plastic-looking faces. Finally, the tutorial details how to install and run the model locally using Comfy UI, including workflows for low-VRAM setups, image-to-image generation, and inpainting.

## Key Points

- Z-Image Base offers significantly higher image variation and supports negative prompts for better control, unlike Z-Image Turbo which often produces repetitive results with the same seed [00:00:53](https://www.youtube.com/watch?v=OtuoOWsnLDg&t=53).
- To run the model on hardware with limited VRAM, users should download compressed GGUF versions and utilize the 'UniLoader GGUF' node within Comfy UI [00:18:22](https://www.youtube.com/watch?v=OtuoOWsnLDg&t=1102).
- While existing LoRAs for Z-Image Turbo are incompatible, the new model is optimized for fine-tuning, and users can quickly generate custom LoRAs from just a few images using the Z-Image Image-to-LoRA tool [00:27:03](https://www.youtube.com/watch?v=OtuoOWsnLDg&t=1623).
