# New open-source "Nano Banana" is here! INSANELY fast

## Executive Summary

The video introduces Flux 2 Klein, a new open-source image generation and editing model that rivals top competitors like Zimage Turbo while running efficiently on consumer hardware with as little as 2GB of VRAM. The presenter conducts head-to-head tests showing Flux 2 Klein excels in image editing consistency and specific artistic styles, though it occasionally struggles with complex anatomy and long text rendering compared to models like Quen Image. Finally, a detailed tutorial demonstrates how to install and run both distilled and quantized versions of the model locally using Comfy UI for free, unlimited offline use.

## Key Points

- Flux 2 Klein offers a unique advantage by combining high-quality image generation and editing capabilities into a single model, eliminating the need to manage separate workflows for different tasks [00:01:05](https://www.youtube.com/watch?v=2OrOufa3eoc&t=65).
- Users with limited hardware can run the model efficiently by utilizing quantized GGUF versions, which allow the software to function on CPUs or GPUs with as little as 2GB of VRAM [00:28:23](https://www.youtube.com/watch?v=2OrOufa3eoc&t=1703).
- For optimal performance, the video recommends using the distilled 4B parameter model for speed and the 9B model for quality, while noting that the 4B variant is licensed under Apache 2 for commercial use [00:02:33](https://www.youtube.com/watch?v=2OrOufa3eoc&t=153).
- The tutorial provides a step-by-step guide to setting up Flux 2 Klein in Comfy UI, including downloading specific text encoders and VAEs, and configuring workflows for both text-to-image and image-to-image editing [00:15:14](https://www.youtube.com/watch?v=2OrOufa3eoc&t=914) [MM:SS](https://www.youtube.com/watch?v=2OrOufa3eoc&t=SECONDS)
