# New #1 open-source AI music generator is INSANE!

## Executive Summary

This video introduces AEP 1.5, a new open-source AI music generator that delivers studio-quality audio across over 50 languages while running efficiently on low VRAM or even CPU-only systems. The presenter demonstrates the tool's capabilities through diverse genre examples and provides a step-by-step tutorial on installing the software via GitHub and configuring its graphical interface. Key features highlighted include the ability to generate full songs in seconds, micro-edit existing tracks using inpainting, and create style-altered cover versions of uploaded audio.

## Key Points

- Install AEP 1.5 locally by cloning the GitHub repository and using the UV package manager to handle dependencies, allowing for free, unlimited, and offline music generation [00:12:15](https://www.youtube.com/watch?v=QzddQoCKKss&t=735).
- Utilize the 'repaint' feature to micro-edit specific sections of an existing song, such as changing lyrics or instrumentals, by uploading source audio and defining the exact time range for modification [00:29:42](https://www.youtube.com/watch?v=QzddQoCKKss&t=1782).
- Leverage the optional language model to enable 'thinking' mode, which plans song structure and expands prompts for more coherent and high-quality outputs, though it can be disabled for lower VRAM usage [00:19:19](https://www.youtube.com/watch?v=QzddQoCKKss&t=1159).
