# New #1 open-source AI music generator is INSANE!

## Executive Summary

This video introduces AEP 1.5, a high-quality open-source AI music generator that outperforms leading closed models like Suno v5 while running efficiently on low VRAM or even CPU. The presenter demonstrates the tool's ability to generate studio-grade tracks across diverse genres and languages, including advanced features like micro-editing existing songs and creating style-specific covers. Finally, a step-by-step tutorial guides viewers through installing the software via GitHub and using its graphical interface to customize generation parameters.

## Key Points

- AEP 1.5 delivers studio-quality audio generation with support for over 50 languages and complex structures, yet remains lightweight enough to run on hardware with less than 4GB of VRAM or standard CPUs [00:56](https://www.youtube.com/watch?v=QzddQoCKKss&t=56).
- Users can leverage the 'repaint' feature to micro-edit specific lyrics or instrumentals within an existing track, or use the 'cover' mode to completely reimagine a song's melody and style while preserving its structure [00:29](https://www.youtube.com/watch?v=QzddQoCKKss&t=1740).
- Installation is streamlined using the UV package manager to clone the repository and automatically download necessary models, allowing for offline, unlimited generation without needing ComfyUI [00:12](https://www.youtube.com/watch?v=QzddQoCKKss&t=735).
