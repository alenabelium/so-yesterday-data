# Elevenlabs just got wrecked. This free AI text to speech is WILD!

## Executive Summary

This video introduces Qwen 3 TTS, a new free and open-source text-to-speech model from Alibaba that rivals paid services like ElevenLabs. It highlights the model's ability to clone voices from just a few seconds of audio, generate entirely new voices via text prompts, and express complex emotions and accents across multiple languages. The presenter demonstrates these capabilities with various examples and provides a step-by-step tutorial on running the tool locally using Comfy UI.

## Key Points

- Qwen 3 TTS offers state-of-the-art voice cloning and generation capabilities, allowing users to create expressive, multilingual audio with specific emotional tones using only a few seconds of reference audio or text prompts [00:00:45](https://www.youtube.com/watch?v=eC8mZceIy5k&t=45).
- The model is lightweight and runs offline on consumer-grade GPUs with as little as 4GB of VRAM, making it a cost-effective alternative to expensive cloud-based TTS services [00:14:29](https://www.youtube.com/watch?v=eC8mZceIy5k&t=869).
- Users can set up the tool by installing a specific Comfy UI workflow, which provides a graphical interface to manage voice cloning, custom pre-built voices, and text-to-voice design nodes [00:18:52](https://www.youtube.com/watch?v=eC8mZceIy5k&t=1132).
