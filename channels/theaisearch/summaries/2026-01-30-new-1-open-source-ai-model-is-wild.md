# New #1 open-source AI model is WILD

## Executive Summary

Kimi K2.5 is a new open-source mixture-of-experts model with 1 trillion total parameters (32B active) that rivals or surpasses top closed models like GPT 5.2, Opus 4.5, and Gemini 3 Pro across multiple benchmarks. Its standout features include multimodal capabilities (solving visual puzzles with autonomous Python code), an Agent Swarm feature that deploys up to 100 agents in parallel for tasks like lead scraping or literature reviews, and specialized agents for slides, documents, spreadsheets, and deep research. Despite its impressive capabilities, the reviewer notes it is more error-prone than GLM 4.7 and requires more prompting iterations, though it has the lowest hallucination rate among top models.

## Key Points

- Kimi K2.5 scores best among open-source models and rivals closed-source leaders on independent benchmarks, costs only $1.1 per million tokens compared to $10 for Opus 4.5, and has the lowest hallucination rate (64% on the extremely challenging Omniscience benchmark) among all top models tested. [21:13](https://www.youtube.com/watch?v=F-ANHe_hr9w&t=1273)
- The Agent Swarm feature deploys up to 100 agents simultaneously, enabling parallel execution of tasks like scraping leads across 7 business categories or synthesizing 40 psychology papers into a 100-page literature review in minutes rather than hours. [09:03](https://www.youtube.com/watch?v=F-ANHe_hr9w&t=543)
- Kimi K2.5's multimodal reasoning allows it to autonomously write and execute Python code to solve complex visual tasks, such as extracting hidden objects from stereograms by reconstructing depth maps or solving the shortest path through a complex maze using computer vision algorithms.
