# LTX 2.3, GPT 5.4, CUDA agent, realtime AI videos, new image models, 360 videos: AI NEWS

## Executive Summary

This AI news roundup covers a packed week including LTX 2.3 (the best open-source video generator with built-in audio), two new open-source image editors (KiwiEdit for video and Fire Red ImageEdit 1.1), ByteDance's CUDA Agent for automatically writing optimized GPU code, and GPT 5.4's release. Other highlights include Helios, a real-time video generator producing 19.5 fps on a single H100; CubeComposer, which converts regular video into 360-degree VR content; Spectrum from ByteDance for 3.5x faster image and video generation; and smaller Qwen 3.5 models that can run on mobile devices.

## Key Points

- ByteDance's CUDA Agent automatically writes, tests, and optimizes GPU kernel code, outperforming top models including Gemini 3 Pro and Opus 4.5 across correctness and speed metrics -- a significant advance for making AI models run faster on GPUs. [18:37](https://www.youtube.com/watch?v=KRE8JqTAEQk&t=1117)
- Fire Red ImageEdit 1.1 is now the best open-source image editor according to benchmarks, surpassing Qwen ImageEdit and even Nano Banana Pro in some tests, with capabilities for face-consistent editing, multi-reference merging, and semantic text-prompted changes. [07:22](https://www.youtube.com/watch?v=KRE8JqTAEQk&t=442)
- LTX 2.3 launched as the best open-source video generator with native audio, supporting up to 20-second 4K videos in vertical format, alongside Alibaba releasing Qwen 3.5 models as small as 0.8B parameters that can run on mobile phones. [28:45](https://www.youtube.com/watch?v=KRE8JqTAEQk&t=1725)
