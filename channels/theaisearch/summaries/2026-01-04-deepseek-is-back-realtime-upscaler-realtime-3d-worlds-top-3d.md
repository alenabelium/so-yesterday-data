# DeepSeek is back, realtime upscaler, realtime 3D worlds, top 3D generator, Qwen Image 2512: AI NEWS

## Executive Summary

This weekly AI news roundup covers several significant releases. DeepSeek published a paper on Manifold Constrained Hyperconnections (MHC), a new architecture that stabilizes multi-lane data flow in deep transformer models, improving both performance and training stability. StreamDiff VSR introduces real-time video upscaling that runs in 0.3 seconds versus thousands of seconds for traditional methods. Other highlights include UltraShape 1.0 for high-fidelity 3D model generation, Ume 1.5 for explorable interactive 3D worlds, IQ Quest Coder V1 achieving near state-of-the-art coding benchmarks at only 40B parameters, and Qwen Image 2512 delivering notably more realistic image generation with superior text rendering.

## Key Points

- StreamDiff VSR enables real-time video upscaling on a single RTX 4090, completing in 0.3 seconds compared to over 4600 seconds for standard diffusion-based upscalers, with code already released on GitHub. [01:16](https://www.youtube.com/watch?v=7kPNA86G_GA&t=76)
- DeepSeek's Manifold Constrained Hyperconnections paper introduces a mathematical framework that stabilizes multi-lane data flow in deep transformers, preventing signal loss across hundreds of layers while outperforming both standard residual connections and unconstrained hyperconnections. [07:13](https://www.youtube.com/watch?v=7kPNA86G_GA&t=433)
- IQ Quest Coder V1, a 40B parameter open-source coding model backed by a Chinese quant trading firm, achieved 76.2% on SWE-Bench Verified after re-evaluation, rivaling models many times its size like Claude 4.5 Opus and Gemini 3 Pro. [24:53](https://www.youtube.com/watch?v=7kPNA86G_GA&t=1493)
