# Best local AI video generator with sound is here!

## Executive Summary

This video reviews LTX 2.3, a new open-source AI video generator that significantly improves motion consistency, audio quality, and anatomical accuracy over its predecessor, LTX 2. The presenter demonstrates these upgrades through various high-action and dialogue-heavy prompts, highlighting new features like native support for vertical formats and first/last frame interpolation. Finally, the video provides a step-by-step tutorial on installing the tool locally using the W2GP platform, enabling users to run the model offline on consumer hardware with as little as 6GB of VRAM.

## Key Points

- LTX 2.3 delivers superior motion consistency and audio clarity compared to LTX 2, effectively reducing warping in high-action scenes and improving lip-sync accuracy for dialogue [01:53](https://www.youtube.com/watch?v=KaQbt6NYrzY&t=113).
- New capabilities in version 2.3 include native support for vertical video formats and the ability to define both start and end frames, though seamless transitions require similar reference images [10:19](https://www.youtube.com/watch?v=KaQbt6NYrzY&t=619).
- Users can run LTX 2.3 locally for free using the W2GP platform, which simplifies installation via a Python virtual environment and optimizes performance for GPUs with as little as 6GB of VRAM [13:28](https://www.youtube.com/watch?v=KaQbt6NYrzY&t=808).
