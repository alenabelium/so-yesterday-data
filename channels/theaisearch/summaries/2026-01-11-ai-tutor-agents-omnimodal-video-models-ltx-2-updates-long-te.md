# AI tutor agents, omnimodal video models, LTX-2 updates, long-term memory, video faceswap: AI NEWS

## Executive Summary

This AI news roundup covers a packed week of releases. Kling's UniVideo is a multimodal unified model that can generate and edit both images and videos from text prompts, reference images, and reference videos in a single model. SimpleMem introduces a smarter long-term memory system for AI agents that compresses, indexes, and selectively retrieves information using far fewer tokens while improving accuracy. Deep Tutor is an open-source AI tutoring agent that proactively teaches and generates personalized practice questions. Other highlights include DreamIDV for video face-swapping, InfiniDepth for ultra-high-resolution depth maps up to 16K, LTX-2 GGUF updates, and Tencent's Hunyuan MT open-source translation model beating much larger competitors.

## Key Points

- SimpleMem is a new memory framework for AI agents that compresses conversations into clean facts, indexes them semantically, and retrieves only what is needed, achieving the highest accuracy with the fewest tokens and fastest retrieval times compared to all existing agent memory systems. - Deep Tutor is an open-source AI tutoring agent that goes beyond simple Q&A by proactively teaching, generating personalized quizzes tailored to skill level, providing visual explanations with diagrams, and supporting deep research across uploaded documents and the web.
