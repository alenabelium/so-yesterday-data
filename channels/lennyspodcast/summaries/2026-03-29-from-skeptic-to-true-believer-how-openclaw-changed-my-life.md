# From skeptic to true believer: How OpenClaw changed my life 

## Executive Summary

Claire Vo, host of the How I AI podcast and three-time CPO, shares her journey from being one of OpenClaw's biggest skeptics (her first install deleted her family calendar) to running nine specialized agents across three Mac Minis. She walks through the practical setup process -- creating a separate machine and Gmail account, using Telegram for communication, and writing identity/soul files -- and demonstrates real use cases including an SDR agent that autonomously sweeps her CRM for enterprise leads, a family logistics agent that coordinates kid pickups and basketball schedules, and a course project management agent. The key insight is that context overload is the main limitation, so splitting work across purpose-built agents (like separate Slack channels) dramatically improves reliability and usefulness.

## Key Points

- The biggest unlock with OpenClaw is running multiple specialized agents rather than one general-purpose assistant; just as you wouldn't put all work communication in a single Slack channel, each agent gets a focused context window -- Sam handles sales, Finn handles family logistics, Howie preps podcast episodes -- which prevents the memory and context overload that frustrates most new users. [40:41](https://www.youtube.com/watch?v=DIa0MYJzM5I&t=2441)
- Claire's sales agent Sam has real economic value: it autonomously sweeps the CRM daily for new PLG signups, enriches company domains using Exa people search to identify decision-makers, sends personalized outreach emails, and manages the pipeline -- replacing 10 hours per week of paid human work she had before. [51:08](https://www.youtube.com/watch?v=DIa0MYJzM5I&t=3068)
- For security and practical isolation, Claire treats each agent like a new employee hire: separate email, delegated calendar access rather than shared passwords, progressive trust building (first read-only email, then drafting, then sending), and explicit anti-prompt-injection instructions in the agent's soul file limiting which communication channels it may accept instructions from. [22:30](https://www.youtube.com/watch?v=DIa0MYJzM5I&t=1350)
