# An AI state of the union: We've passed the inflection point & dark factories are coming

## Executive Summary

Simon Willison, co-creator of Django and coiner of the term "prompt injection," delivers a comprehensive state of the union on AI-assisted software development. He explains that the November 2025 inflection point -- when GPT 5.1 and Claude Opus 4.5 crossed a quality threshold -- fundamentally changed what's possible, with professional engineers now producing 95% of their code through AI agents. Simon distinguishes between "vibe coding" (non-programmers building prototypes without reading code) and "agentic engineering" (professionals using coding agents for production software), and explores the emerging "dark factory" pattern where companies like StrongDM build and test software without any human reading the code, using swarms of simulated users for QA. He warns that the industry is headed for a "Challenger disaster" moment due to increasingly unsafe use of these systems, while arguing that experienced engineers are more valuable than ever because these tools amplify existing skills.

## Key Points

- The November 2025 inflection point occurred when models went from "most of the time it mostly works" to "almost all of the time it does what you told it to do" -- a threshold crossing that transformed coding agents from curiosities into core engineering tools; Simon now writes most of his code from his phone while walking his dog. [03:15](https://www.youtube.com/watch?v=wc8FBhQtdsA&t=195)
- The "dark factory" pattern, pioneered by StrongDM, involves policies where nobody writes or reads code; instead, quality is ensured through swarms of AI-simulated end users running 24/7 (at $10K/day in tokens), testing the software in simulated versions of Slack, Jira, and Okta that the coding agents themselves built from API documentation. [12:49](https://www.youtube.com/watch?v=wc8FBhQtdsA&t=769)
- Using coding agents well is mentally exhausting and takes every inch of senior experience; Simon can run four agents in parallel but is wiped out by 11 AM, and he warns of gambling-like addiction patterns where engineers lose sleep keeping agents running -- the universal skill needed now is agency, the human ability to decide what problems to take on. [27:34](https://www.youtube.com/watch?v=wc8FBhQtdsA&t=1654)
