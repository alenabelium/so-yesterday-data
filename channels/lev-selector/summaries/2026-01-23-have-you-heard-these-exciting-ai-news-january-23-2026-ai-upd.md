# Have you heard these exciting AI news? - January 23, 2026 AI Updates Weekly

## Executive Summary

This weekly update emphasizes the shift from manual coding to orchestrating AI agent fleets. Skills.sh now hosts over 6,500 reusable AI skills that load on demand for under 50 tokens each, compared to 5,000-10,000 tokens for MCP server tools. The episode covers Claude Code's rapid growth to 3 million VS Code downloads in days, Cursor's experiment producing 3 million lines of code using organized agent teams (planners, workers, judges), and Baidu's Ernie model reaching production with 2.4 trillion parameters. A Microsoft research paper predicts 97 million AI-related jobs will emerge to offset 85 million displaced roles, with AI-skilled workers earning 56% more.

## Key Points

- Skills.sh provides a directory of 6,574 reusable AI skills as lightweight markdown files (under 50 tokens per header), loaded on demand -- dramatically cheaper than MCP servers which typically consume 5,000-10,000 tokens per interaction. [00:45](https://www.youtube.com/watch?v=cZYFF1zvI8k&t=45)
- Cursor ran hundreds of AI coding agents simultaneously for weeks, producing over 3 million lines of code; agents organized as planners, workers, and judges built a functional web browser from scratch in under a week, plus a Windows 7 emulator and Excel clone each spanning over a million lines. [21:28](https://www.youtube.com/watch?v=cZYFF1zvI8k&t=1288)
- Microsoft research predicts that while 85 million jobs will be outsourced to AI, 97 million AI-related roles will emerge; by 2027, 70% of job skills globally are expected to change and workers with AI skills will earn roughly 56% more than those without. [28:05](https://www.youtube.com/watch?v=cZYFF1zvI8k&t=1685)
