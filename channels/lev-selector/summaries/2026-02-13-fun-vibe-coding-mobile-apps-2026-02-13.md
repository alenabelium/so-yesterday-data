# Fun Vibe-coding Mobile Apps - 2026-02-13

## Executive Summary

Guest Joe demonstrates building a sophisticated iOS mood/energy tracker app called Triilog entirely through vibe coding, starting with the Roar platform and later switching to Claude Code with Xcode for Apple-specific features like iCloud storage. He never read or wrote a single line of code, yet produced a feature-rich app now live in the App Store with a Pomodoro timer, habit tracking, calorie counting, and weather correlation. His key insight is that near-zero implementation cost enables rapid iteration, allowing him to reach "version 50" ideas he never could have conceived upfront -- the true revolution is not building the first version but iterating endlessly at no cost.

## Key Points

- Joe built a complete iOS app published to the App Store using only vibe coding on Roar ($20/month) and Claude Code ($100/month), never writing or even reading the generated code; the app includes mood tracking, habit tracking, calorie counting, Pomodoro timer, and weather correlation.
