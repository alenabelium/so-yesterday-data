# Have you heard these exciting AI news? - February 13, 2026 AI Updates Weekly

## Executive Summary

This episode's dominant story is Open Claw, the fastest-growing GitHub project in history, now being negotiated for acquisition by Meta and OpenAI. Created by Peter Steinberger from Vienna, it evolved from a simple WhatsApp relay into a modular autonomous agent platform that Chinese giant Baidu has already integrated for its 700 million users. The episode introduces "skills in the middle" as a new programming paradigm where system architects become skills architects, writing markdown descriptions instead of code. Other highlights include Johns Hopkins researchers achieving 100x compression for LoRA adapters, MiniMax 2.5 as a cheap open-source coding model, and a developer automating his entire daily workflow through Claude Code skills including LinkedIn, Gmail, and YouTube management.

## Key Points

- Open Claw grew from a personal WhatsApp relay to 214,000+ GitHub stars, the fastest-growing project in GitHub history; Meta's Mark Zuckerberg and OpenAI top management are personally negotiating to acquire it, while Baidu has already integrated it for 700 million users in China. [04:32](https://www.youtube.com/watch?v=GsHA0eAcmaI&t=272)
- The "skills in the middle" paradigm replaces hard-coded program logic with markdown skill files that guide AI agent execution -- system architects become skills developers writing text descriptions instead of code, with skills loaded on demand and sourced from community repositories. [18:50](https://www.youtube.com/watch?v=GsHA0eAcmaI&t=1130)
- A developer named Chris automated his entire daily workflow through Claude Code skills running on a dedicated Mac Mini for weeks, including video research, YouTube uploads, LinkedIn messaging, Gmail, and GitHub operations -- skills that took 6 minutes to learn initially execute in 40 seconds on subsequent runs.
