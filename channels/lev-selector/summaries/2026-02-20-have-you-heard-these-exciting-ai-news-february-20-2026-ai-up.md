# Have you heard these exciting AI news? - February 20, 2026 AI Updates Weekly

## Executive Summary

The headline story is a developer who replicated Google's Notebook LM by extracting its 214 skills into markdown files and running them locally with Claude Code, demonstrating that even core AI services from major companies can now be replaced by personal skill-based agents. Peter Steinberger (Open Claw creator) joined OpenAI, which created a separate open-source foundation for him, while Anthropic banned subscription users from running Open Claw due to excessive token consumption -- pushing users to OpenAI or local models. The episode also covers Claude Sonet 4.6 with programmatic tool calling, XAI's Grok 4.2 with its "society of mind" four-agent architecture, and Claude Code's new agent teams feature for collaborative multi-agent development.

## Key Points

- A developer replicated Google's Notebook LM by extracting 214 unique skills into markdown files and running them locally with Claude Code, gaining more flexibility (e.g., outputting PowerPoint instead of PDF) and proving that AI services from major companies can be replaced by personal skill-based agents in 2-3 days. [00:02](https://www.youtube.com/watch?v=BT02OEDY6H8&t=2)
- Anthropic banned subscription users from running Open Claw and similar autonomous agents due to excessive token consumption, forcing users to pay per token via API; Peter Steinberger joined OpenAI which created a separate open-source foundation, making OpenAI subscriptions the new default for Open Claw users. [05:40](https://www.youtube.com/watch?v=BT02OEDY6H8&t=340)
- A "beginner vs expert" prompting technique asks AI to write about a topic at beginner level, then expert level, then explain the conceptual shifts between them -- revealing the assumptions, priorities, and tradeoffs that distinguish expertise, which can then be fed back as a refined prompt for even better results.
