# How to Set Up Claude Code in 2026 (Beginner Tutorial)

## Executive Summary

This beginner tutorial walks through the complete setup of Claude Code in 2026, from installation to building a working kanban to-do app. The video covers choosing between a Claude subscription (recommended for fixed costs) and API keys, configuring terminal settings, using planning mode to design features with clarifying questions, and running Claude Code in both the terminal and the VS Code extension. It also introduces advanced workflows like using deep-dive sub-agents to review code quality in separate context windows.

## Key Points

- A Claude subscription ($20/month) is strongly recommended over API keys because it caps your monthly cost -- no matter how much you use Claude Code, you never exceed the subscription fee, whereas API costs can spiral quickly. [02:09](https://www.youtube.com/watch?v=kddjxKEeCuM&t=129)
- Planning mode (shift+tab) lets you have a conversation with Claude to design your project before any code is written, and the agent asks valuable clarifying questions that can surface requirements you hadn't considered. [09:34](https://www.youtube.com/watch?v=kddjxKEeCuM&t=574)
- Deep-dive sub-agents can be spun up in a separate session to review the implementation for bugs -- in the demo, three background explore agents found four critical and four medium-priority bugs that the main coding agent had missed. [16:34](https://www.youtube.com/watch?v=kddjxKEeCuM&t=994)
