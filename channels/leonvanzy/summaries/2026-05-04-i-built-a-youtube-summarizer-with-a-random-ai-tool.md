# I Built a YouTube Summarizer with a RANDOM AI Tool

## Executive Summary

This video demonstrates building a functional YouTube summarizer using Cursor's Composer 2 agent, starting with a random tool selection challenge. The creator scaffolds a Next.js application, installs specific skills for UI design and AI integration, and leverages the agent's ability to perform end-to-end testing in an embedded browser. By utilizing the AI SDK for structured outputs and OpenRouter for model inference, the project successfully generates TLDRs, key takeaways, and timestamped moments from video transcripts.

## Key Points

- The project begins by spinning a wheel to select Cursor as the AI tool, followed by scaffolding a Next.js app and installing skills for shadcn UI components and front-end design to create the initial interface [00:01:35](https://www.youtube.com/watch?v=xA7DG6QAVtc&t=95).
- Cursor's Composer 2 agent is tasked with retrieving YouTube transcripts using a Node.js library and performs autonomous end-to-end testing by interacting with the application in the integrated browser [00:08:20](https://www.youtube.com/watch?v=xA7DG6QAVtc&t=500).
- The final phase involves integrating the AI SDK and OpenRouter to process transcripts into structured outputs like TLDRs and key moments, with the agent verifying the complete workflow through self-testing [00:13:23](https://www.youtube.com/watch?v=xA7DG6QAVtc&t=803).
