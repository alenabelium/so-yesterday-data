# I Hired 14 AI Agents to Build a SaaS Product (Paperclip)

## Executive Summary

The video demonstrates how to use Paperclip, an open-source framework, to orchestrate a team of 14 specialized AI agents that function as a fully automated software company. Unlike direct prompting tools where the user acts as the project manager, Paperclip allows the user to act as a board member who delegates high-level vision to a CEO agent, which then autonomously hires and manages sub-agents for engineering, marketing, and QA. The tutorial walks through deploying the system on a VPS, configuring agent roles and skills, and integrating the workflow with GitHub for version control and deployment.

## Key Points

- Deploy Paperclip on a VPS using Docker to ensure your AI agent team runs 24/7 independently of your local machine [00:06:00](https://www.youtube.com/watch?v=XCamx18L7XE&t=360)
- Shift your role from active project manager to board member by assigning high-level vision tasks to a CEO agent that autonomously delegates work to specialized sub-agents [00:03:22](https://www.youtube.com/watch?v=XCamx18L7XE&t=202)
- Integrate the agent team with a GitHub repository by generating and adding SSH deploy keys, enabling the agents to commit code and create pull requests automatically [00:20:50](https://www.youtube.com/watch?v=XCamx18L7XE&t=1250)
