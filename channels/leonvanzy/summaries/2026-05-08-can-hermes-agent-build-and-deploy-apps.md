# Can Hermes Agent Build and Deploy Apps?

## Executive Summary

This video demonstrates how to configure the Hermes AI agent to function as a fully autonomous coding assistant capable of building and deploying web applications. The presenter guides viewers through setting up a VPS, integrating the Vercel CLI for deployment, and installing specific agent skills to handle project creation and version control. By sending a single prompt via Telegram, the agent successfully scrapes data, generates a portfolio website, and deploys it to a live production URL. The experiment concludes by showing the agent's ability to update the deployed code and redeploy changes automatically, proving its viability for rapid prototyping and personal projects.

## Key Points

- The video outlines the initial setup process, including deploying Hermes on a Hostinger VPS, configuring OpenAI models, and connecting the agent to Telegram for remote access [00:01:23].
- Viewers are shown how to authenticate the agent with Vercel by installing the CLI tool and securely setting environment variables to enable automated project deployment [00:04:37].
- The final test involves prompting Hermes to build a personal portfolio page, which it successfully creates, deploys to a live URL, and updates with new content upon request [00:09:47] [MM:SS](https://www.youtube.com/watch?v=gxx39SE0aDw&t=SECONDS)
