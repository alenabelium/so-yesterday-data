# Claude Code Agent View: Parallel Agents Are Here

## Executive Summary

Claude Code introduces 'Agent Views,' a new feature that allows developers to dispatch and manage multiple parallel AI agent sessions from a single dispatcher interface. This update eliminates the need for multiple terminal windows by enabling background tasks, such as security audits or continuous improvement loops, to run simultaneously. The video demonstrates how to create, monitor, and control these sessions, including advanced workflows involving work trees for parallel design experimentation.

## Key Points

- Agent Views allow users to dispatch multiple Claude Code sessions via a central dispatcher or by running 'claude -d' in separate terminals, all viewable in one dashboard [00:02:00](https://www.youtube.com/watch?v=tJVUAzLZUyI&t=120).
- Users can configure permission modes (e.g., YOLO/dangerous) and utilize advanced commands like 'loop' for scheduled improvements or 'goal' for complex, long-running tasks [00:05:07](https://www.youtube.com/watch?v=tJVUAzLZUyI&t=307).
- The feature supports cross-project management and integrates with Git work trees to parallelize design variations, allowing developers to test multiple UI implementations simultaneously [00:09:06](https://www.youtube.com/watch?v=tJVUAzLZUyI&t=546).
