# Consensus AI + Claude Code = No More Hallucinations

## Executive Summary

Leon van Zyl shows how to integrate the Consensus AI MCP server with Claude Desktop and Claude Code to ground AI responses in peer-reviewed research papers instead of web searches. He demonstrates two practical uses: research-backed planning mode in Claude Code, where the agent researches actual studies to inform app feature design (building a sleep optimization app), and embedding the Consensus MCP directly into an application's chat interface so end users get research-cited answers. The free Consensus MCP provides access to over 200 million research papers and can be added to any MCP-compatible agent.

## Key Points

- Standard Claude responses cite blog posts and paywalled articles from Google searches, while the Consensus MCP tool provides answers grounded in actual peer-reviewed scientific papers with clickable citations linking directly to the research. [00:00](https://www.youtube.com/watch?v=fyRRafCurzE&t=0)
- Research-backed planning in Claude Code uses the Consensus MCP to inform application design with real science -- the sleep app demo pulled in studies on chronotypes, circadian rhythms, and CBT-I therapy, producing features the developer would never have known to include. [04:36](https://www.youtube.com/watch?v=fyRRafCurzE&t=276)
- The Consensus MCP can be embedded directly into applications using the AI SDK with OAuth authentication, enabling end-user chat interfaces that answer questions exclusively from peer-reviewed research rather than the model's training data. [09:32](https://www.youtube.com/watch?v=fyRRafCurzE&t=572)
