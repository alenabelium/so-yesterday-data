# Claude's New Agent Harness Runs Claude for Hours

## Executive Summary

This video introduces Anthropic's new 'Managed Agents' feature, a fully hosted infrastructure that allows Claude to run autonomous, long-duration coding tasks without consuming local resources. The presenter demonstrates how to configure agents, environments, and sessions using the Console, CLI, and SDK to build a 'Ship It' application that generates software on demand. By leveraging built-in prompt caching and secure remote execution, developers can delegate complex projects like building a Trello clone to the agent while it iterates and adds features over hours or days.

## Key Points

- Developers can utilize the Managed Agents Console to prototype and configure agent definitions, including models, system prompts, and tools, before exporting the configuration for production use [00:04:32](https://www.youtube.com/watch?v=DpfLbBuhHOg&t=272).
- The system architecture separates the user interface from the heavy lifting by running the agent in a remote Anthropic environment, allowing for secure code execution and file manipulation without local infrastructure costs [00:02:42](https://www.youtube.com/watch?v=DpfLbBuhHOg&t=162).
- To ensure the agent has sufficient context for complex tasks, developers should scrape official documentation into markdown files within the project folder to serve as a reference index for the agent [00:13:05](https://www.youtube.com/watch?v=DpfLbBuhHOg&t=785).
