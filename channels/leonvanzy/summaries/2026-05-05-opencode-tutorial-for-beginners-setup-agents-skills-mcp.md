# OpenCode Tutorial for Beginners: Setup, Agents, Skills & MCP

## Executive Summary

This video provides a comprehensive tutorial on OpenCode, an open-source AI coding tool that serves as a powerful alternative to proprietary solutions like Claude Code. The presenter guides viewers through the installation process, connecting to various LLM providers, and leveraging 'skills' to enhance agent capabilities for specific frameworks like Next.js. Key advanced features covered include configuring memory files for strict agent behavior, utilizing sub-agents for parallel task execution, and integrating MCP servers for automated browser-based testing.

## Key Points

- OpenCode is a free, open-source terminal-based AI coding tool that supports multiple LLM providers and allows users to install specialized 'skills' to enhance agent performance for specific tasks like UI design and framework implementation [00:00:00](https://www.youtube.com/watch?v=uZGDO0L-Dr4&t=0).
- Users can enforce strict behavioral rules and project context by creating an agents.md memory file, which helps manage the agent's reasoning effort and ensures consistent adherence to design systems [00:10:44](https://www.youtube.com/watch?v=uZGDO0L-Dr4&t=644).
- The tutorial demonstrates how to configure MCP servers, such as Playwright, to enable the AI agent to autonomously navigate a browser, test the application, and visually confirm that UI changes are working correctly [00:29:14](https://www.youtube.com/watch?v=uZGDO0L-Dr4&t=1754).
- Advanced workflows involve using sub-agents to delegate tasks in parallel, which protects the main agent's context window from becoming too large and degrading performance [00:12:52](https://www.youtube.com/watch?v=uZGDO0L-Dr4&t=772).
