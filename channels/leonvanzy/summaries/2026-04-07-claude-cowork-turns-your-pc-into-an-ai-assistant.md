# Claude Cowork Turns Your PC Into an AI Assistant

## Executive Summary

Leon van Zyl walks through Claude Cowork, Anthropic's desktop AI assistant that can interact with local files, browse the web, connect to external services like Gmail, and be controlled remotely from a mobile device. The tutorial demonstrates practical use cases including downloading stock images from the web to organized local folders and setting up an automated email responder for a restaurant using custom skills, scheduled tasks, and the Gmail connector. The video highlights Cowork's project system for organizing context and rules, its skill creation workflow for teaching the agent domain-specific behavior, and the dispatch feature for controlling the desktop assistant from a phone.

## Key Points

- Claude Cowork can interact with local files, browse the web via a Chrome extension, and organize results into folder structures on your machine, but it requires explicit permission before accessing any folder and can only work within directories you approve. [05:47](https://www.youtube.com/watch?v=YGP706Fcggo&t=347)
- The skill creation system allows you to teach Claude domain-specific knowledge and behavior -- such as restaurant menu details, operating hours, and response tone -- which the agent then uses to autonomously draft email replies to customer inquiries via the Gmail connector. [14:16](https://www.youtube.com/watch?v=YGP706Fcggo&t=856)
- Scheduled tasks and the dispatch feature enable fully automated workflows: you can set Claude Cowork to respond to emails hourly or daily, and control everything remotely from your phone as long as the desktop stays awake. [21:21](https://www.youtube.com/watch?v=YGP706Fcggo&t=1281)
