# Claude Code Just Changed How I Code Forever (Remote Control)

## Executive Summary

Claude Code's new remote control feature lets you control local Claude Code sessions from your mobile phone via the Claude iOS or Android app. You start a session by running `claude remote control` in your project folder, scan a QR code, and then send prompts, approve permissions, and monitor progress from anywhere. Sessions sync bidirectionally with the Claude Desktop app, and the code stays local on your machine -- no cloud deployment required.

## Key Points

- Setting up remote control requires only running `claude remote control` in your project folder and scanning the QR code with the Claude mobile app -- the connection appears in the Code tab and supports all standard Claude Code features. [00:00](https://www.youtube.com/watch?v=u7gx6_LXdnI&t=0)
- Code remains local on your machine rather than being deployed to the cloud, giving you the flexibility to work on local projects remotely while maintaining the same access to tools, skills, and permissions you would have at your desk. [02:00](https://www.youtube.com/watch?v=u7gx6_LXdnI&t=120)
- Sessions sync between mobile and the Claude Desktop app -- renaming a session on desktop immediately reflects on mobile, and the full conversation history is accessible from both interfaces. [02:00](https://www.youtube.com/watch?v=u7gx6_LXdnI&t=120)
