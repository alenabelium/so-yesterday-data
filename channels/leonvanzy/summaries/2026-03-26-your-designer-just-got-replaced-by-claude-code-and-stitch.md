# Your Designer Just Got Replaced by Claude Code and Stitch

## Executive Summary

The video demonstrates a workflow where Google Stitch is used to rapidly prototype high-fidelity UI designs before integrating them into a codebase via Claude Code. By connecting Stitch as an MCP server, developers can instruct AI agents to generate design systems, create screens, and export HTML directly into their projects. This approach solves the common issue of AI coding agents producing poor visual results by separating the design iteration phase from the implementation phase.

## Key Points

- Connect Google Stitch to Claude Code via an MCP server configuration to enable the agent to programmatically generate and manage UI designs without manual intervention [00:06:06](https://www.youtube.com/watch?v=gfmbH7aU3TU&t=366).
- Export the generated design system as a reference file and instruct your coding agents to enforce these specific tokens and styles when building components [00:11:40](https://www.youtube.com/watch?v=gfmbH7aU3TU&t=700).
- Instruct Claude Code to download the raw HTML from Stitch screens rather than relying on screenshots, ensuring animations and layout metadata are preserved during implementation [00:15:27](https://www.youtube.com/watch?v=gfmbH7aU3TU&t=927).
