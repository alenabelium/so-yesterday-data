# Daily Digest — March 27, 2026

The 5 most important AI developments from recent content.

## 1. Launch of the First Tradable Compute Price Index (OCPI)

The introduction of the OR Compute Price Index on the Bloomberg terminal establishes a standardized, tradable benchmark for GPU compute, treating it as a commodity similar to electricity. This breakthrough enables institutional investors to hedge risks and allocate capital efficiently, potentially unlocking trillions in funding stalled by pricing opacity.

Sources: [The First Tradable Compute Price Index](/videos/2UxEvsWfDns)

## 2. Strategic Pivot to AGI: OpenAI Shuts Down Sora for 'Spud' Model

OpenAI has officially discontinued its video generation model, Sora, to reallocate all resources toward the development of its 'Spud' AGI model and a foundation focused on Alzheimer's research. This marks a decisive industry shift from generative media experiments to high-velocity autonomous intelligence and scientific problem-solving.

Sources: [Welcome to March 25, 2026](/videos/GcydRleZP8U)

## 3. Autonomous Code Skills: Self-Building and Self-Testing Agents

Claude Code Skills have evolved to allow AI agents to autonomously generate, test, and evaluate custom skills using blind A/B comparisons against baseline performance. This capability enables developers to create highly specialized, self-optimizing workflows that significantly outperform static base models while reducing token waste.

Sources: [Claude Code Skills Now Build AND Test Themselves](/videos/epZy_NajGnA)

## 4. The K-Shaped AI Job Market and the Rise of 'Specification Precision'

The labor market has fractured into a K-shaped split where traditional roles stagnate while specialized AI architects command salaries up to $400K due to a shortage of talent skilled in 'specification precision' and failure pattern recognition. Success now depends on defining exact agent behaviors and context architectures rather than basic prompting.

Sources: [The AI Job Market Split in Two. One Side Pays $400K and Can't Hire Fast Enough.](/videos/4cuT-LKcmWs), [Your AI Resume Is Worthless. Here's What Hiring Managers Actually Look For.](/videos/qh-KUDWZGaU)

## 5. Integration of Design and Code via Google Stitch and Claude Code

A new workflow has emerged where Google Stitch acts as an MCP server for Claude Code, allowing AI agents to generate high-fidelity UI designs and export them directly into codebases with preserved metadata. This solves the historical limitation of coding agents producing poor visual results by separating design iteration from implementation.

Sources: [Your Designer Just Got Replaced by Claude Code and Stitch](/videos/gfmbH7aU3TU)
