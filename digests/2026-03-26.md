# Daily Digest — March 26, 2026

The 5 most important AI developments from recent content.

## 1. Launch of the First Tradable Compute Price Index (OCPI)

The introduction of the OR Compute Price Index on the Bloomberg terminal establishes a transparent, tradable benchmark for GPU compute, treating it as a commodity similar to electricity. This development is critical for unlocking trillions in institutional capital by enabling risk management through derivatives and futures contracts, solving a major bottleneck in global AI infrastructure investment.

A major update to Claude Code Skills now allows AI agents to automatically generate, build, and test custom skills using blind A/B comparisons against baseline performance. This capability, combined with dynamic context injection and isolated 'fork' sessions, enables developers to create highly specialized, self-validating workflows that significantly reduce token waste and latency.

Sources: [Claude Code Skills Now Build AND Test Themselves](/videos/epZy_NajGnA)

## 5. Integration of Google Stitch and Claude Code for Autonomous UI Design

A new workflow connects Google Stitch as an MCP server to Claude Code, allowing AI agents to autonomously prototype high-fidelity UI designs and export them directly into codebases. This integration solves the historical weakness of coding agents in visual design by separating the iteration phase from implementation, enabling end-to-end autonomous frontend development.
